## рџ“Љ Technical Indicators Used

| Indicator | Usage |
|---|---|
| **RSI (14)** | < 30 = oversold (buy), > 70 = overbought (sell) |
| **MACD** | EMA12 - EMA26; positive = bullish momentum |
| **SMA 20 & 50** | Golden cross (SMA20 > SMA50) = bullish |
| **Volatility** | Annualised standard deviation of returns |
| **Momentum** | Today's % change as a directional signal |

Signal strength combines all indicators into a score в†’ STRONG BUY / BUY / HOLD / SELL / STRONG SELL
